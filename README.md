# Deploy

This project is hosted on **Vercel**. It is connected to Github repository, so simply push changes to GitHub to update site.

```shell
git add .
git commit -m "message"
git push
```

# Initial Deployment

```shell
git init
git remote add origin <url>
git fetch
git reset origin/master
```
